# Kat's training camp ⭐️

A cute personal dashboard for workouts, motivation, and weekly consistency.

This is a simple static HTML/CSS/JavaScript project built to keep my weekly training schedule, mindset reminders, and habit tracker in one place.

## ✨ Features

- Weekly training schedule
- Workout locations
- Schedule and motivation tabs
- Random mantra generator (_hardcoded_)
- Weekly star tracker
- Progress saved locally with `localStorage`
- Mobile-friendly design

## 🏃‍♀️ Weekly Structure

- 3 runs
- 2 HYROX sessions
- 3 gym sessions
- 1 fitboxing session
- Optional social padel or chill swimming

## ⭐️ Weekly Star Tracker

The tracker is designed to mark small wins across the week:

- Strength
- Running
- HYROX
- Nutrition
- Sleep & recovery

Each category has one star per day. Clicking a star fills it in, and the progress is saved in the browser using `localStorage`.

## 💗 Mindset

The goal is not perfection.  
The goal is consistency, confidence, strength, and learning to show up like an athlete who respects herself.
